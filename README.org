* Requirements
- GNU Emacs
- GNU Make
- C++ Compiler
  - GNU Compiler Collection (g++)
  - LLVM Clang Compiler (clang++)

In Debian GNU/Linux:
#+BEGIN_SRC shell
sudo apt install emacs-nox g++ make
#+END_SRC

In Mac OS X:
#+begin_src shell
xcode-select --install
#+end_src
