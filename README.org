#+TITLE: C++ for AtCoder

* C++ for AtCoder 

** Requirements
- GNU Emacs (Optional)
- C++ Compiler
  - GNU Compiler Collection (g++)
  - LLVM Clang Compiler (clang++)
- GNU Make (make)
- CMake (cmake)

In Debian GNU/Linux:
#+BEGIN_SRC shell
$ sudo apt install cmake g++ make
$ sudo apt install emacs # or emacs-nox
#+END_SRC


** Libraries
- C++ Standard Library (C++14)
- Boost (1.60)
