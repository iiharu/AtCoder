#+TITLE: C++ for AtCoder

* C++ for AtCoder 

** Requirements
- Artistic Style (astyle)
- GNU Emacs (Optional)
- C++ Compiler
  - GNU Compiler Collection (g++)
  - LLVM Clang Compiler (clang++)
- GNU Make (make)

In Debian GNU/Linux:
#+BEGIN_SRC shell
$ sudo apt install astyle emacs g++ make
#+END_SRC
