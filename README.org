* Requirements
- Artistic Style
- GNU Emacs (Optional)
- C++ Compiler
  - GNU Compiler Collection (g++)
  - LLVM Clang Compiler (clang++)
- GNU Make

In Debian GNU/Linux:
#+BEGIN_SRC shell
sudo apt install astyle emacs-nox g++ make
#+END_SRC
