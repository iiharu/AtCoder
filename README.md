
# C++ for AtCoder

## 注意
- モチベーションを維持するためにコードをアップするレポジトリ.
- コンテスト中にアップはしない.
- コードは自分で解いたものと解説を見て解いたものがごちゃごちゃ

## Requirements
- C++ Compiler
  - GNU Compiler Collection (`g++`)
  - LLVM Clang Compiler (`clang++`)

### Optional Requirements
- CMake (`cmake`)
- GNU Make (`make`)
